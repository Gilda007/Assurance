📌 Présentation générale

Qu'est-ce que LOMETA ?

LOMETA (Logiciel de Management et d'Évaluation des Transports Automobiles) est une plateforme complète qui permet aux professionnels de l'assurance automobile de gérer l'ensemble de leurs opérations de manière centralisée et efficace.

🎯 Objectifs du logiciel

🏦
Centralisation
Toutes vos données d'assurance rassemblées en un seul endroit : véhicules, contrats, clients, paiements et documents.
⚙️
Automatisation
Calcul automatique des primes, génération des documents et suivi des échéances sans intervention manuelle.
🔐
Sécurité
Protection des données sensibles, authentification sécurisée et certification des modules pour garantir l'intégrité.

👥 Profils utilisateurs

🏢
Compagnies
Gestion des contrats, suivi des primes, gestion des sinistres et reporting.
🏪
Agences
Création de devis, émission de contrats, suivi client et gestion des paiements.
🚛
Gestionnaires de flottes
Supervision des véhicules d'entreprise, gestion des polices groupées et reporting.
👤
Particuliers
Gestion de ses véhicules, consultation des contrats et suivi des paiements.
💡 Astuce : LOMETA est conçu pour s'adapter à votre profil. Les fonctionnalités disponibles varient selon votre rôle (administrateur, agent, gestionnaire de flotte, etc.).

🔑 Connexion et interface

Écran de connexion

Pour accéder à LOMETA, saisissez vos identifiants (email et mot de passe) dans l'écran de connexion. Si vous êtes un nouvel utilisateur, votre administrateur vous fournira vos accès.

🔐 Sécurité : Si vous avez oublié votre mot de passe, utilisez le lien "Mot de passe oublié" pour recevoir un email de réinitialisation.

🖥️ Interface principale

Une fois connecté, vous arrivez sur le Tableau de bord qui vous donne une vue d'ensemble de votre activité :

📊 Le tableau de bord affiche :

  • 📈 Statistiques clés : Nombre de véhicules, contrats actifs, clients, primes totales, sinistres en cours.
  • 📋 Activités récentes : Les dernières actions effectuées (création de contrat, paiement, modification de véhicule).
  • 🔔 Notifications : Alertes sur les échéances proches, les contrats expirant, les paiements en attente.
  • 📊 Graphiques : Évolution des primes, répartition des garanties, tendances des sinistres.

🧭 Navigation

La barre latérale gauche vous permet d'accéder aux différents modules :

  • 🏠 Tableau de bord - Vue d'ensemble et indicateurs clés
  • 🚗 Véhicules - Gestion du parc automobile
  • 📄 Contrats - Gestion des polices d'assurance
  • 👥 Clients - Gestion des contacts et clients
  • 🚛 Flottes - Gestion des flottes de véhicules
  • 💰 Paiements - Suivi des transactions et échéances
  • 📊 Rapports - Génération de rapports et statistiques
  • ⚙️ Paramètres - Configuration du compte et de l'application
⌨️ Raccourcis : Utilisez les raccourcis clavier pour naviguer plus rapidement : Ctrl+D pour le tableau de bord, Ctrl+F pour la recherche, F5 pour rafraîchir.

🚗 Gestion des véhicules

Le module Véhicules est le cœur de LOMETA. Il permet d'enregistrer, de modifier et de suivre l'ensemble des véhicules assurés.

Ajouter un véhicule

Pour ajouter un nouveau véhicule dans la base :

  1. Cliquez sur le bouton "+ Nouveau véhicule"
  2. Remplissez les informations obligatoires :
    • 🔴 Immatriculation (obligatoire) - Plaque d'immatriculation du véhicule
    • 🔴 Marque et Modèle (obligatoires) - Ex: Toyota Hilux
    • Châssis - Numéro de série (VIN)
    • Année - Année de mise en circulation
    • Puissance fiscale - En CV (chevaux fiscaux)
    • Énergie - Essence, Diesel, Électrique, Hybride
    • Nombre de places - Capacité du véhicule
    • Valeur à neuf - Prix d'achat du véhicule
    • Valeur vénale - Valeur actuelle estimée
    • Catégorie - VP (Particulier), VU (Utilitaire), PL (Poids Lourd), etc.
    • Zone tarifaire - A, B ou C selon la localisation
  3. Cliquez sur "Enregistrer" pour valider
💡 Astuce : Si vous avez plusieurs véhicules à ajouter, utilisez la fonction d'importation en masse depuis un fichier Excel ou CSV.

🔍 Rechercher un véhicule

Utilisez la barre de recherche en haut de l'écran pour retrouver rapidement un véhicule par :

  • Immatriculation (recherche partielle possible)
  • Marque et modèle
  • Nom du propriétaire
  • Numéro de châssis

📋 Fiche détaillée du véhicule

En cliquant sur un véhicule dans la liste, vous accédez à sa fiche détaillée qui regroupe :

  • 📋 Informations générales : Immatriculation, marque, modèle, année, énergie, puissance, places
  • 🛡️ Garanties : Liste des garanties souscrites avec leurs montants
  • 📄 Contrat : Détails du contrat d'assurance en cours
  • 💰 Finances : Historique des paiements, échéances, solde restant
  • 📊 Statistiques : Évolution des primes, sinistres, bonus/malus
  • 📁 Documents : Tous les documents associés (Carte Rose, attestation, etc.)
⚠️ Important : Les informations du véhicule sont utilisées pour le calcul automatique des primes. Assurez-vous de les maintenir à jour (valeur vénale, date de mise en circulation, etc.).

🚗 Gestion des véhicules (suite)

📥 Importer des véhicules en masse

Pour importer plusieurs véhicules simultanément :

  1. Cliquez sur le bouton "Importer"
  2. Téléchargez le modèle Excel fourni
  3. Remplissez les colonnes avec vos données
  4. Importez le fichier pour créer automatiquement tous les véhicules
📋 Le modèle Excel comprend :
  • Immatriculation (obligatoire)
  • Marque et Type (obligatoire)
  • Genre, Puissance en CV, Usage, Nombre de Places
  • Valeur Neuve, Valeur Venale, PMEC
  • Détails des garanties (RC/RTI, Vol, Incendie, DTA, etc.)
  • Primes et réductions

📊 Suivi du parc automobile

Le module Véhicules vous permet de suivre l'évolution de votre parc avec des indicateurs clés :

  • 🚗 Nombre total de véhicules - Vue d'ensemble du parc
  • 📅 Répartition par année - Âge moyen du parc
  • 🔋 Répartition par énergie - Essence, Diesel, Électrique
  • 🏷️ Répartition par catégorie - VP, VU, PL, etc.
  • 📄 Statut des contrats - Véhicules assurés vs non-assurés

📈 Indicateur clé : Le taux de renouvellement des contrats vous permet d'identifier les véhicules dont l'assurance arrive à échéance et de planifier les renouvellements.

📁 Documents associés

Pour chaque véhicule, vous pouvez gérer les documents suivants :

  • 📄 Contrat d'assurance - PDF de la police
  • 🛡️ Attestation - Attestation d'assurance
  • 📊 Devis - Devis initial ou de renouvellement
  • 📋 Carte Rose - Document officiel d'assurance
  • 🎫 Vignette - Vignette à apposer sur le pare-brise
  • 🧾 Quittance - Preuve de paiement
⚠️ Attention : La suppression d'un véhicule est définitive et supprime également ses contrats et documents associés. Utilisez plutôt la fonction "Désactiver" si le véhicule n'est plus assuré.

📄 Gestion des contrats

Le module Contrats permet de gérer l'ensemble des polices d'assurance, de la souscription au renouvellement.

Créer un contrat

La création d'un contrat se fait en quelques étapes :

  1. Sélectionnez le véhicule à assurer
  2. Choisissez la compagnie d'assurance
  3. Définissez la période de validité (date de début et date de fin)
  4. Sélectionnez les garanties souhaitées
  5. Indiquez les frais supplémentaires (accessoires, ASAC, carte rose, vignette)
  6. Validez pour générer le numéro de police et le contrat
💡 Astuce : Le contrat peut être créé à partir d'un devis existant. Le numéro de police est attribué automatiquement selon les paramètres de la compagnie.

📊 Suivi des contrats

Chaque contrat dispose d'une fiche détaillée avec :

  • 📋 Informations générales : Numéro de police, compagnie, périodes, statut
  • 🛡️ Garanties souscrites : Détail des garanties et leurs montants
  • 💰 Récapitulatif financier : Prime brute, réduction, prime nette, frais, PTTC
  • 💳 Historique des paiements : Toutes les transactions effectuées
  • 📅 Échéancier : Calendrier des paiements à venir
  • 📄 Documents : Tous les documents associés (contrat, avenants, etc.)

🔄 Renouvellement automatique

LOMETA gère le renouvellement des contrats de manière automatique :

  • 📅 30 jours avant l'échéance : Une notification est envoyée
  • 📅 15 jours avant l'échéance : Un rappel est affiché dans le tableau de bord
  • 📅 Jour de l'échéance : Le contrat est automatiquement renouvelé si la case "Renouvellement automatique" est cochée
  • 📅 Après l'échéance : Le contrat passe en statut "En attente" si non renouvelé
⚠️ Important : Le renouvellement automatique n'engage pas automatiquement le paiement. Vous devez valider les conditions et procéder au paiement pour que le nouveau contrat soit effectif.

🛡️ Gestion des garanties

Le module Garanties vous permet de configurer et de suivre l'ensemble des garanties proposées pour les contrats d'assurance automobile.

📋 Types de garanties

🛡️
RC - Responsabilité Civile
Garantie obligatoire qui couvre les dommages causés à des tiers par le véhicule assuré. C'est la base de tout contrat d'assurance automobile.
⚖️
DR - Défense et Recours
Protège l'assuré en cas de litige, en prenant en charge les frais de défense juridique suite à un accident.
🚗
VOL - Vol du véhicule
Indemnise l'assuré en cas de vol total ou partiel du véhicule. Le montant dépend de la valeur vénale du véhicule.
🔫
VB - Vol à main armée
Garantie spécifique pour les vols avec violence, offrant une protection renforcée contre ce type de sinistre.
🔥
Incendie
Couvre les dommages causés par un incendie, qu'il soit d'origine accidentelle ou criminelle.
🪟
Bris de glaces
Prend en charge le remplacement des vitres et pare-brise endommagés, sans franchise dans la plupart des cas.
🔧
AR - Assistance Réparation
Garantie qui couvre les frais de réparation en cas de panne ou d'accident, avec une assistance 24h/24.
💥
DTA - Dommages Tous Accidents
Garantie complète qui couvre tous les dommages au véhicule, quel que soit le responsable de l'accident.
👥
IPT - Individuelle Personnes Transportées
Protège les passagers transportés par le véhicule, en cas de dommages corporels.

⚙️ Configuration des garanties

Pour chaque contrat, vous pouvez sélectionner les garanties souhaitées et configurer :

  • Montant de la garantie : Valeur assurée pour chaque garantie
  • Réductions : Pourcentage de réduction applicable (ex: -10% sur RC, -50% sur Vol)
  • Franchises : Montant restant à la charge de l'assuré en cas de sinistre
  • Options spécifiques : Personnalisation selon les besoins du client
💰 Conseil : Le calcul des primes est automatique. Vous pouvez appliquer des réductions en masse sur les garanties sélectionnées via la fonction "Réduction en masse" dans le tableau des véhicules.
⚠️ Important : La RC (Responsabilité Civile) est obligatoire. Les autres garanties sont optionnelles mais fortement recommandées pour une protection complète.

🚛 Gestion des flottes

Le module Flottes permet de regrouper et de gérer plusieurs véhicules appartenant à une même entreprise ou organisation.

📋 Qu'est-ce qu'une flotte ?

Une flotte est un ensemble de véhicules gérés de manière groupée. Cela permet de :

  • Centraliser les informations de tous les véhicules
  • Simplifier la gestion des contrats d'assurance
  • Optimiser les coûts avec des tarifs préférentiels
  • Suivre globalement les échéances et les paiements
  • Générer des rapports consolidés par flotte

🚀 Créer une flotte

  1. Cliquez sur "+ Nouvelle flotte"
  2. Saisissez le nom et le code de la flotte
  3. Sélectionnez le propriétaire (le contact associé)
  4. Choisissez l'assureur (compagnie d'assurance)
  5. Définissez la période de validité
  6. Ajoutez les véhicules à la flotte
  7. Validez pour créer la flotte
💡 Astuce : Vous pouvez importer une flotte complète depuis un fichier Excel. Utilisez la fonction "Importer" pour charger plusieurs véhicules en une seule fois.

📥 Importer une flotte

L'importation de flotte est une fonctionnalité avancée qui permet de :

  • Charger un fichier Excel ou CSV contenant tous les véhicules
  • Calculer automatiquement les garanties pour chaque véhicule
  • Appliquer des réductions en masse sur les garanties
  • Générer les contrats pour tous les véhicules de la flotte
  • Vérifier les données avant l'importation définitive

📋 Le tableau d'importation affiche : Imm. matriculation, Marque, Type, Genre, Puissance, Usage, Places, PMEC, Valeurs, Détail des garanties, Primes, et une colonne d'actions pour modifier chaque véhicule.

📊 Suivi des flottes

La fiche détaillée d'une flotte vous permet de :

  • Voir tous les véhicules de la flotte
  • Gérer les contrats de chaque véhicule
  • Suivre les échéances globales
  • Générer des rapports consolidés
  • Exporter les données de la flotte

👥 Gestion des clients

Le module Clients (ou Contacts) permet de gérer l'ensemble des personnes et entreprises liées à vos contrats d'assurance.

📋 Qui sont les clients ?

Un client peut être :

  • Un particulier : Personne physique assurant son véhicule personnel
  • Une entreprise : Personne morale assurant ses véhicules de société
  • Un gestionnaire de flotte : Professionnel gérant plusieurs véhicules
  • Un courtier : Intermédiaire entre l'assuré et la compagnie

➕ Ajouter un client

Pour créer un nouveau client dans la base :

  1. Cliquez sur "Nouveau client" ou "Nouveau contact"
  2. Remplissez les informations d'identité :
    • Nom et prénom (ou raison sociale)
    • Date de naissance (pour les particuliers)
    • Nationalité
    • Numéro contribuable (si disponible)
    • Catégorie socio-professionnelle
  3. Ajoutez les coordonnées :
    • Téléphone (principal et portable)
    • Email (principal et secondaire)
    • Adresse postale et ville
    • Numéro de permis de conduire
  4. Précisez le type de client et le statut VIP si applicable
  5. Attribuez un chargé de clientèle
  6. Validez pour enregistrer le client

📊 Fiche détaillée du client

La fiche client regroupe toutes les informations et activités liées à ce contact :

  • 👤 Informations personnelles : Identité, coordonnées, documents
  • 📄 Contrats : Tous les contrats d'assurance du client
  • 🚗 Véhicules : Tous les véhicules associés au client
  • 🚛 Flottes : Les flottes dont le client est propriétaire
  • 📎 Documents : Tous les documents liés au client
  • 📊 Statistiques : Primes totales, nombre de contrats, historique
💡 Astuce : Utilisez les badges "Type de client" et "Statut VIP" pour identifier rapidement vos clients prioritaires et adapter votre suivi.

💰 Paiements et échéances

Le module Paiements vous permet de suivre l'ensemble des transactions financières liées aux contrats d'assurance.

💳 Enregistrer un paiement

Pour enregistrer un paiement sur un contrat :

  1. Ouvrez la fiche du contrat concerné
  2. Dans la section "Paiements", cliquez sur "Nouveau paiement"
  3. Saisissez le montant versé
  4. Choisissez le mode de paiement :
    • Espèces
    • Carte bancaire
    • Virement
    • Chèque
    • Orange Money
    • MTN Mobile Money
  5. Ajoutez une note si nécessaire
  6. Validez pour enregistrer le paiement
📋 Automatisation : Un reçu est automatiquement généré pour chaque paiement. Vous pouvez l'imprimer ou l'envoyer par email au client.

📊 Suivi des échéances

LOMETA vous aide à ne jamais manquer une échéance de paiement :

  • 📅 Visualisation : Calendrier des paiements à venir
  • 🔔 Notifications : Alertes 30 jours, 15 jours et 7 jours avant l'échéance
  • 📊 État des paiements : Statut des paiements (en attente, partiel, complet)
  • 💳 Solde restant : Affichage clair du montant restant à payer

📊 Indicateur clé : Le taux de recouvrement vous permet de mesurer l'efficacité de votre gestion des paiements et d'identifier les retards.

📄 Documents financiers

Pour chaque paiement, vous pouvez générer :

  • 🧾 Reçu de paiement : Preuve de la transaction
  • 📊 Quittance : Document officiel de paiement de la prime
  • 📋 Relevé de compte : Historique complet des transactions
  • 📄 Facture : Pour les paiements professionnels
⚠️ Important : Un contrat non payé à échéance est automatiquement suspendu. Assurez-vous de valider les paiements dans les délais pour maintenir la couverture d'assurance.

📁 Documents et impressions

LOMETA intègre un module complet de génération et d'impression de documents pour tous vos besoins administratifs.

🖨️ Types de documents disponibles

📄
Contrat d'assurance
Document complet de la police avec toutes les garanties et conditions. Format PDF imprimable.
🛡️
Attestation
Attestation d'assurance officielle à fournir en cas de contrôle. Contient les informations essentielles du contrat.
📋
Carte Rose
Document officiel à conserver dans le véhicule. Format papier A4 avec toutes les mentions légales.
🎫
Vignette
Vignette à apposer sur le pare-brise. Format adapté avec logo et numéro de police.
🧾
Quittance
Preuve de paiement de la prime d'assurance. Utile pour justifier le règlement.
📊
Devis
Devis détaillé avec toutes les garanties, leurs montants et les conditions. Parfait pour les propositions commerciales.

📤 Impression et export

Pour chaque document, vous avez plusieurs options :

  • 🖨️ Impression directe : Envoyer le document vers votre imprimante
  • 📄 Export PDF : Sauvegarder le document au format PDF
  • 📧 Envoi par email : Envoyer le document directement au client
  • 📥 Téléchargement : Télécharger le document sur votre ordinateur
💡 Astuce : Utilisez l'impression groupée pour générer plusieurs documents en une seule fois. Sélectionnez plusieurs véhicules et choisissez le document à imprimer.

📋 Gestion des documents

Tous les documents générés sont automatiquement archivés dans le système et accessibles à tout moment :

  • 📁 Historique : Tous les documents générés pour chaque véhicule
  • 🔍 Recherche : Retrouvez un document par date, type ou numéro de police
  • 📤 Export : Téléchargez les documents en lot ou individuellement
  • 🗑️ Gestion : Supprimez les documents obsolètes

📋 Note : Les documents sont générés avec vos informations de compagnie (logo, adresse, coordonnées). Personnalisez les modèles dans les paramètres de l'application.

📊 Rapports et statistiques

Le module Rapports vous permet d'analyser en profondeur vos données et de prendre des décisions éclairées.

📈 Tableaux de bord

Le tableau de bord principal vous offre une vue d'ensemble de votre activité avec :

  • 📊 Indicateurs clés (KPI) : Nombre de véhicules, contrats actifs, primes totales, clients, sinistres
  • 📈 Graphiques : Évolution des primes, répartition des garanties, tendances des sinistres
  • 📋 Activités récentes : Dernières actions effectuées dans le système
  • 🔔 Notifications : Alertes sur les échéances et événements importants

📊 Rapports personnalisés

Vous pouvez générer des rapports personnalisés selon vos besoins :

📄 Rapports de véhicules
Liste complète des véhicules avec toutes leurs caractéristiques, garanties et statut des contrats.
📄 Rapports de contrats
Récapitulatif des contrats en cours, expirés, à renouveler, avec les primes et les paiements.
📄 Rapports clients
Analyse de la clientèle : répartition par type, ancienneté, primes totales, fidélité.
📄 Rapports financiers
Suivi des paiements, échéances, impayés, avec des indicateurs de performance.
📄 Rapports de flottes
Analyse consolidée par flotte : nombre de véhicules, primes totales, performance.
📄 Rapports de sinistres
Historique des sinistres, fréquence, coût moyen, tendances par type de garantie.

📤 Export des rapports

Tous les rapports peuvent être exportés dans différents formats :

  • 📄 PDF : Pour une impression professionnelle
  • 📊 Excel : Pour une analyse poussée des données

🔐 Certification des modules

La certification des modules est un mécanisme de sécurité qui garantit l'intégrité et l'authenticité des modules installés dans LOMETA.

📋 Qu'est-ce qu'un module certifié ?

Un module certifié est un module qui a été signé numériquement par une autorité de confiance (LOMETA ou un éditeur certifié). La certification garantit que :

  • ✅ Intégrité : Le module n'a pas été modifié après sa certification
  • ✅ Authenticité : Le module provient bien de l'éditeur annoncé
  • ✅ Sécurité : Le module ne contient pas de code malveillant
  • ✅ Compatibilité : Le module est compatible avec votre version de LOMETA

🔍 Comment vérifier la certification d'un module ?

Pour vérifier qu'un module est certifié :

  1. Ouvrez le dossier du module dans l'interface de LOMETA
  2. Recherchez le fichier certificate.json à la racine du module
  3. Vérifiez la présence du badge ✅ Certifié dans l'interface
  4. Consultez les détails du certificat :
    • Nom du module
    • Version
    • Certifié par (LOMETA ou l'éditeur)
    • ID de certification
    • Date de certification
🔐 Sécurité : Un module certifié affiche un badge vert dans l'interface. En cas de doute, vous pouvez vérifier manuellement la signature du certificat.

🔧 Gestion des certificats

LOMETA intègre un gestionnaire de certificats qui vous permet de :

  • 📋 Voir la liste de tous les certificats émis
  • 🔍 Vérifier la validité d'un certificat
  • ❌ Révoquer un certificat (si le module est compromis)
  • 📄 Exporter les certificats pour audit

💡 Bon à savoir : La certification des modules est optionnelle mais fortement recommandée pour garantir la sécurité de votre système. Les modules certifiés sont prioritaires lors des mises à jour.

📋 Commandes de certification

Pour les administrateurs, voici les commandes disponibles :

  • Setup : Configurer les clés de certification
  • Certify : Certifier un module (--module /chemin)
  • Verify : Vérifier un module certifié (--module /chemin)
  • List : Lister tous les certificats émis
  • Revoke : Révoquer un certificat (--id CERT_ID)
⚠️ Important : N'installez jamais un module non certifié sur votre système. Les modules non certifiés peuvent compromettre la sécurité de vos données.
